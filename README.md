# update-fetch-derivations

Update some fetchFromGitHub usages in a nix source file.

## Usage

```
update-fetch-derivations
    Update GitHub fetch revisions and hashes for a given nix source file.

    Example:

      update-fetch-derivation fetch-from-github.nix
```
